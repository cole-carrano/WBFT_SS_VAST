# WBFT_SS_VAST
##Stock Synthesis runs testing the inclusion of new VAST indices for western bluefin tuna

## Runs Guide
* **Orig** = "Covariate" Model. Original stock synthesis run from most recent (2021) assessment.
* **Run9** = "No Env." Model. Same as "orig"/"covariate", but with the AMO influence removed
* **Run8** = "US only VAST". Includes US-only VAST index instead of US_RR_GT177 index
* **Run9** = "Joint VAST". Includes US/Can joint VAST index instead of US_RR_GT177 index and CAN_SWNS indices
